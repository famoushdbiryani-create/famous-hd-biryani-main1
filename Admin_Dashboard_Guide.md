## 2. Using PostImages.org for Menu Pictures

Because direct image uploading can be expensive, we have integrated a fast, free, and easy way to add images to your menu items using **PostImages.org**.

**How to upload an image:**
1. Go to [https://postimages.org/](https://postimages.org/)
2. Click on **Choose images** and select the food picture from your computer or phone.
3. Once the image finishes uploading, you will see a list of links.
4. Locate the row that says **Direct link**.
5. Click the **copy icon** right next to the Direct link (it should end in `.jpg`, `.png`, or `.webp`).
6. Go back to your Admin Dashboard and **paste that link** into the "Image URL" field when adding or editing a menu dish.

## 3. Managing the Menu (Add, Update, Hide, Remove)

You can manage your entire menu in real-time from the **Menu Management** section on the left sidebar.

### Adding a New Item
1. Click the **"+ Add New Dish"** button at the top of the Menu Management page.
2. Fill in the required details:
   - **Name:** The name of the dish (e.g., *Chicken 65*).
   - **Price:** The price without the dollar sign (e.g., *12.99*).
   - **Category:** Select the appropriate category (e.g., *Appetizers*).
   - **Dietary:** Choose Veg, Non-Veg, Vegan, etc.
   - **Description:** A short, mouth-watering description of the food.
   - **Badge (Optional):** Add "Chef's Special", "Spicy", or "Bestseller".
   - **Image URL:** Paste the *Direct link* you copied from PostImages.org.
3. Click **Save Menu Dish**.

### Updating an Existing Item
1. Scroll down or use the Search Bar to find the item you want to edit.
2. Click the **Edit** (pencil icon) button on that item.
3. The details will load into the form at the top. Change whatever you need (price, description, name, etc.).
4. Click **Save Menu Dish**.

### Hiding an Item (Out of Stock)
If you run out of an item for the day but don't want to delete it permanently:
1. Find the item in the list.
2. Look for the **"Show on Site"** toggle switch (checkbox).
3. **Uncheck** it. The badge will change to **[HIDDEN]**. 
4. The item will temporarily disappear from the public website once you publish. Check the box again when it's back in stock!

### Removing/Deleting an Item
1. Find the item you want to permanently delete.
2. Click the **Delete** (trash can icon) button.
3. Confirm the deletion. **Warning:** This cannot be undone!

## 4. Creating Active Offers

You can show flashy, time-sensitive offers on the website's homepage to drive more sales! Go to the **Offers & Promo** section.

**How to add a new offer:**
1. Click **"+ Create New Offer"**.
2. Fill in the details:
   - **Offer Title:** E.g., *Weekend Biryani Special*
   - **Discount Text:** E.g., *15% OFF* or *Buy 1 Get 1 Free*
   - **Icon:** Select an icon that matches the offer (like a Tag, Star, or Gift).
   - **Valid From & Valid Until:** Select the start and end dates/times for the offer. (The offer will automatically disappear from the website when it expires!)
   - **Description:** Details or promo codes (e.g., *Use code WEEKEND15 at checkout*).
   - **Banner Text:** This text will scroll continuously at the very top of your website as an announcement bar.
3. Click **Save Offer**. It will go live immediately!

---

## 5. Publishing Your Changes

The Admin Dashboard has a "Staging" feature. This means when you edit the Menu, Home Page, About Page, or Contact Info, it is saved as a **Draft**. You can safely make a bunch of changes without the public seeing them yet!

**How to preview and publish:**
1. Click the **PREVIEW SITE** button at the top right of your dashboard. This opens a private tab where you can see exactly how your changes will look.
2. Once you are happy with how everything looks, look at the top right of your dashboard header.
3. Click the bright green **PUBLISH NOW** button (with the rocket icon).
4. The system will sync all your drafts to the live website instantly!

*(Note: Changes made to the Gallery and Offers are published instantly and do not require the Publish button).*

---

### Need Help?
If you ever run into issues where the Admin Panel seems out of sync with the website, click the **UPDATE WEBSITE TO ADMIN PANEL** button in the header. This will pull the current live website data back into your dashboard drafts so you can start fresh!
