How to Upload Food Images (PostImages.org)

The website uses a free image hosting service called PostImages.org to display photos of your food items. This is fast, free, and very easy to use. Follow these steps every time you need to add a new food photo.

1
Open a new browser tab and go to https://postimages.org/
2
Click the large "Choose Images" button in the center of the page.
3
Select your food photo from your computer or phone. Wait a few seconds for it to upload.
4
After uploading, you will see a results page with several different link types listed.
5
Find the row that says "Direct link". It is the link that ends in .jpg, .png, or .webp.
6
Click the copy button (clipboard icon) next to the Direct link. The link is now copied.
7
Go back to your Admin Dashboard and paste (Ctrl+V or Cmd+V) the link into the "Image URL" field.

⚠️ Important

Only use the Direct Link from PostImages.org — not the "Hotlink for forums", "Thumbnail link", or "Viewer link". The Direct link is the one that ends directly in .jpg or .png.

Famous HD Biryani Menu Management
3

Managing Your Menu

The Menu Management section (click it in the left sidebar) is where you add, edit, hide, or remove food dishes on your website. All changes are saved as a draft until you hit "Publish Now".

✅ Adding a New Dish

1
Click "+ Add New Dish" at the top of the Menu Management page.
2
A form will appear. Fill in all the details (see the table below).
3
Click "Save Menu Dish". The item is now saved as a draft.
4
Click the green "PUBLISH NOW" button in the top header to push it to the live website.
Field Name What to Enter Required?
Name The dish name (e.g., Chicken 65, Hyderabadi Biryani) Required
Price Number only, no $ sign (e.g., 12.99) Required
Category Select from dropdown: Quick Bites, Biryani, etc. Required
Dietary Select Veg / Non-Veg / Vegan / Gluten-Free Required
Description A short appetizing description of the food item Optional
Badge Special label: Chef's Special, Bestseller, Spicy, New Optional
Image URL Paste the Direct Link from PostImages.org here Optional
Show on Site Checkbox — keep it checked so the dish appears on website Required

⚡ Quick Actions for Existing Items

✏️

Edit / Update Item

Click the pencil icon on any dish. Change the price, name, description, or image. Then click "Save Menu Dish".

👁️

Hide Item (Out of Stock)

Find the item and uncheck the "Show on Site" checkbox. The dish stays saved but disappears from the public menu.

🗑️

Delete Item

Click the trash can icon on the dish. Confirm the deletion popup. Warning: This is permanent and cannot be undone!

🔍

Find an Item Quickly

Use the Search Bar at the top of the Menu section to filter by name, category, or dietary type.

Famous HD Biryani Offers & Publishing

Creating Promotional Offers

You can create time-limited offers that automatically appear on your homepage with a scrolling announcement banner. Offers go live immediately when saved and automatically disappear on their expiry date.

1
Click "Offers & Promo" in the left sidebar, then click "+ Create New Offer".
2
Fill in all the offer details (see table below).
3
Click "Save Offer". The offer will go live on the website immediately!
Field Name What to Enter Required?
Offer Title Short headline (e.g., Weekend Biryani Special) Required
Discount Text The deal value (e.g., 15% OFF or Buy 1 Get 1 Free) Required
Icon Select an emoji/icon that fits the offer (Tag, Star, Gift, etc.) Optional
Valid From The date and time the offer starts (e.g., today's date) Required
Valid Until The date and time the offer automatically expires Required
Description More details or promo code (e.g., Use code BIRYANI15) Optional
Banner Text This text scrolls in the announcement bar at the top of the website Optional

Previewing & Publishing to the Live Website

When you save changes to Menu items, Home page, About page, or Contact info, they are saved as a private draft first. The public website is not changed yet. You can preview all your changes before going live, and only push them live when you are happy.

Make Changes

Edit menu items, prices, descriptions in the Admin Dashboard

→
👁️
Preview Site

Click "PREVIEW SITE" to see how it looks before it goes live

→
🚀
Publish Now

Click the green "PUBLISH NOW" button at the top to go live

✅ Where is the Publish Now Button?

Look at the top-right corner of your admin dashboard — you will see a bright green "PUBLISH NOW" button with a rocket icon (🚀). This button is always visible no matter which section you are in. One click publishes all your changes at once.

ℹ️ Note on Instant vs. Draft Changes

Draft (requires Publish Now): Menu item changes, Home page edits, About page, Contact info

Instant (no publish needed): Offers & Promotions, Gallery photos — these go live immediately when saved.

🔄 Something Looks Out of Sync?

If the admin panel is showing old or incorrect data that doesn't match your website, click the "UPDATE WEBSITE TO ADMIN PANEL" button in the top header. This refreshes the admin with the current live website data.
